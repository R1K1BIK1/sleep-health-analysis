# Анализ сна и образа жизни

## Описание проекта
Анализ влияния образа жизни на качество сна с использованием Python и Yandex DataLens.

## Технологии
- Python (pandas, matplotlib, seaborn)
- Jupyter Notebook
- Yandex DataLens
- GitHub

## Структура проекта
- `sleep_analysis.ipynb` - полный анализ в Jupyter
- `Sleep_health_and_lifestyle_dataset.csv` - исходные данные
- `README.md` - это описание
